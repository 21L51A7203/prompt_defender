# Prompt Defender ⚔️

Detect if your AI prompts contain prompt injection!

🧠 Built using Python and Streamlit  
👩‍💻 Created by Ruchita Nandini

## Features
- Detects malicious prompt patterns
- Clean vs. Injected scoring system
- Safety tips and suggestions
- Graphical score bars
- Real-time analysis without reloading
